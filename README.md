### Network Defense: ACL, DHCP Relay/Helper, OSPF, VLAN, Trunking, and Port Security Project


This project presents the planning and implementation of a secure enterprise network architecture focused on resilience, segmentation, and controlled access. 
The design integrates VLAN segmentation and trunking to isolate broadcast domains, OSPF for dynamic and scalable routing, and ACLs to enforce least privilege policies across network boundaries.
DHCP Relay enables centralized IP management across multiple subnets, while Port Security mitigates unauthorized device access at the edge.
From a business risk perspective, the solution reduce lateral movement, limits the attack surface, and improves fault isolation, directly impacting service continuity and compliance.
The result is a structured, policy driven network that balances performance, scalability, and security, demonstrating practical alignment between technical controls and organizational risk management.
